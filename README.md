# Assitente-para-declara-o-de-Imposto-de-Renda
Este projeto visa fornecer uma ferramenta para facilitar a vida de quem precisa realizar a declaração do imposto de renda. Ela possibilita a centralização das principais informações solicitadas na declaração e alguns campos calculados para facilitar durante o levantamento de montantes e em quais bancos os valores se encontram.
